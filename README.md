# CSS Transform Animation with Collision Detection

This repository contains a simple JavaScript animation demonstrating CSS transform effects applied to moving boxes with collision detection. The animation is implemented using HTML5 Canvas API.

## Features

- Moving boxes with different transform effects such as translate, rotate, scale, skew, and matrix.
- Collision detection between boxes and canvas borders.
- Reflection animation upon collision with borders.


## Customization

- You can modify the number of boxes, their initial properties, and the canvas size in the `script.js` file to customize the animation.
- To change the appearance of the boxes, modify the CSS styles in the `style.css` file.
